Using pathlib to check if a path exists before creating a directory like this: * Path(os.path.join(sys.path[0], "../../ToolDirectory/{}".format(TOOL_DIR_NAME))).mkdir(parents=True, exist_ok=True) Returns this error: TypeError: mkdir() got an unexpected keyword argument 'exist_ok' exist_ok is a valid parameter for pathlib with python3.8 After an hour of research and fiddling, the python2.7 solution to import pathlib2 instead of just pathlib makes exist_ok work as expected.
